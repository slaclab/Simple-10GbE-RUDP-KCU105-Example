How to run the Software in Interactive Mode¶

  1. Setup rogue software (refer to Rogue Software Setup)

  2. Run the interactive python script with the -i argument

    $ cd Simple-10GbE-RUDP-KCU105-Example/software
$ ipython -i -- scripts/interactive.py

  3. You now have an interactive command terminal to the rogue software.

  4. As an example, let’s change the ScratchPad from 0xdeadbeef to 0x12345678

    In [1]: root.Core.AxiVersion.ScratchPad.getDisp()
Out[1]: '0xdeadbeef'

In [2]: root.Core.AxiVersion.ScratchPad.setDisp(0x12345678)

In [3]: root.Core.AxiVersion.ScratchPad.getDisp()
Out[3]: '0x12345678'
