How to the Rogue FileWriter and FileReader

Rogue includes a general purpose file write and file reader to quickly write data to disk and analysis offline.

  1. Start up the PyDM gui (refer to How to run the Software Development GUI with KCU105 Hardware)

  2. Go to the System tab (A), click on Auto Name (B), then click on Open (C)

# Go to the Debug Tree tab and nagivate to Root.App.AppTx and execute 0x100 for SendFrame

  1. Go to the System tab wait for the frame count to reach 256 (A), and click on Close (B)

  2. Close the PyDM GUI

  3. Run the fileReader python script

    $ python scripts/fileReader.py --dataFile data_20220614_092954.dat
